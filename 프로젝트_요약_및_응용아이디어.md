## 추천 응용 프로젝트

### 1순위 — 운동 자세 코치 앱
**사용 프로젝트:** 5번 (Pose) + 8번 (MOT)

```
핵심 기술: 관절 각도 계산 → 자세 점수화
예시: 팔꿈치 각도 = arccos(벡터 내적) → 스쿼트/푸쉬업 횟수 카운팅
추가 기능: ID 추적으로 여러 명 동시 측정
```

- 어깨-팔꿈치-손목 각도 계산 → 운동 폼 피드백
- 반복 횟수 자동 카운트 (각도 임계값 기반)
- 여러 사람 동시 코칭 (BoT-SORT ID 유지)

### 2순위 — 실시간 배경 교체 / 크로마키
**사용 프로젝트:** 9번 (Instance Segmentation)

```python
# 핵심 코드 아이디어
mask_bool = (mask_resized > 0.5)
frame[~mask_bool] = background_img[~mask_bool]  # 배경만 교체
```

- 줌 가상 배경처럼 사람만 남기고 배경 교체
- 배경에 영상/이미지 합성
- 클래스별로 오브젝트만 추출 (e.g., 사람만, 컵만)

### 3순위 — 시각 보조 / 사물 음성 안내

```python
import pyttsx3
tts = pyttsx3.init()
```

### 4순위 — 매장 / 공간 동선 분석
**사용 프로젝트:** 10번 (Crowd Density) + 8번 (MOT)

```
밀도맵: 어느 구역에 사람이 많이 머물렀는지 시각화
궤적: 고객 이동 경로 패턴 분석
활용: 매장 진열대 배치 최적화, 혼잡 구역 파악
```

- CSV 저장 기능 추가 → 시간대별 분석 가능

### 5순위 — AR 거리 측정 / 공간 인식
**사용 프로젝트:** 6번 (Depth) + 7번 (YOLO-World)

```python
# 감지된 물체의 깊이값 읽기
cx, cy = (x1+x2)//2, (y1+y2)//2
distance = depth_map[cy, cx]  # 상대적 거리값
cv2.putText(frame, f"{distance:.2f}m", ...)
```

- 화면 속 물체까지의 거리 표시
- 가까운 물체 경고 (거리 임계값)
- 실내 공간 3D 구조 파악
